Easily add a customizable Social media link to the footer of your WordPress site, enhancing social media engagement.

## Description

The Social Media Footer Link Plugin allows you to easily add a customizable Social media link to the footer of your WordPress site. This plugin is perfect for bloggers and businesses who want to increase their social media engagement by directing visitors to their Social media page.

### Features
- **Customizable Social media Link**: Set your Social media profile or page URL.
- **Adjustable Link Color**: Choose the color of the link to match your site's design.
- **Optional Display in Posts Feed**: Choose whether to show the link in single posts, pages, or the main feed.
- **Enable/Disable**: Easily enable or disable the plugin without losing settings.

## Installation

1. Upload the `Social media-footer-link` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Navigate to Settings > Social media Footer Link to configure the plugin.

## Usage

1. Go to the plugin settings page under Settings > Social media Footer Link.
2. Enter your Social media profile or page URL.
3. Set the desired link color using a hex code.
4. Choose whether to display the link in posts feed.
5. Enable or disable the plugin as needed.

## Frequently Asked Questions

### How do I find my Social media profile or page URL?
To find your Social media URL, navigate to your Social media profile or page, and copy the URL from the address bar.

### Can I change the color of the link?
Yes, you can set any color for the link using a hex code in the plugin settings.

### Will this plugin slow down my site?
The Social media Footer Link plugin is lightweight and designed to minimally impact your site's performance.
